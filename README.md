"# js_pig_game" 
Two players are playing the roll-dice game. Whoever got a score of 100 first, wins the game
rulls: 
1. New game starts with player1, player1 rolls dice first. Any dice number other than 1 will add to the player's current score 
2. If the dice number is 1, the current score is cleared to 0 and player will switch 
3. When the hold buttone is clicked, the points add to the player's total score, and the player switchs to anther. 
4. To start a new game, the new game button is click, every score resets to 0 and palyer1 is the defualt player
